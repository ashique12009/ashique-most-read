== Description ==

Show most read posts in your home page or anywhere you want.

== Installation ==

1. Go to plugins then activate the plugin.
2. Use this shotcode: [show_most_read_posts] or in your PHP script use like this <?php echo do_shorcode('[show_most_read_posts]'); ?>

You will see a unordered list with this class name: ashique-most-read-posts-list
You can change the list markup by CSS with that class name.
