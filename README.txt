Show most read posts in frontend. With the help of settings you can customize how many posts you want to show.

== Description ==

Show most read posts into your frontend with the help of shortcode. 

The shortcode is: [show_most_read_posts]

You can show it by shortcode, moreover you can get the query result from transient value from this key: most_read_posts.

You can get settings, from admin main menu settings -> Most read posts settings.

From settings, you can set how many posts you want to show.

More from settings, you can set how many days interval you need to see your posts.
